Cognos Dashboard Work Flow:
---------------------------------------------------------------------------------
1. Complete the objectives
2. Export the objectives as CSV
3. Upload it to Cognos Analytics
4. Create DashBoard from the CSV's
5. Make a Report with Dashboard
6. Forecast the future Results.

Node RED work Flow:
--------------------------------------------------------------------------------
1. Create UI
2. Grab values from the UI
3. Set global variables to UI
4. Get Access Token with HTTP request
5. Send the values to scoring end point along wiht access token using http request node
6. Get the predicted output
7. Parse the output
8. Finally display the output on UI
